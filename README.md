# 📌 Project Overview

This project is a **classic Brick Breaker arcade game** developed using **Python and Pygame**.

The player controls a paddle to bounce the ball and break bricks placed at the top of the screen. Each brick has different strength levels and disappears once its power reaches zero.

The objective is to destroy all bricks without letting the ball fall below the paddle.

Game Features:

- Real-time collision detection
- Multiple brick strength levels
- Paddle movement control
- Ball physics simulation
- Win and Game Over conditions
- Restart option

---

# 🧠 How It Works

1. The game window is created using **Pygame display module**
2. Bricks are generated in rows with different power levels
3. A ball object moves with defined velocity
4. Collision detection checks interaction with:
   - Walls
   - Paddle
   - Bricks
5. Brick power decreases when hit
6. The player wins when all bricks are destroyed
7. Game over occurs when the ball touches the bottom

# ⚙ Installation

## 🪟 Windows

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/python-brick-breaker-game.git
```

### 2. Go to the project folder

```bash
cd python-brick-breaker-game
```

### 3. Install required packages

```bash
pip install pygame
```

### 4. Run the program

```bash
python brick_breaker_game.py
```

# 🎮 Game Controls

| Key | Action |
|----|----|
| ← Left Arrow | Move Paddle Left |
| → Right Arrow | Move Paddle Right |
| Mouse Click | Start / Restart Game |

---

# 🧑‍💻 Usage

1. Run the Python script  
2. Click anywhere on the screen to start the game  
3. Use **left and right arrow keys** to control the paddle  
4. Prevent the ball from falling below the paddle  
5. Destroy all bricks to win the game  

Game Messages:

- **YOU WON!** → All bricks destroyed
- **GAME OVER!** → Ball falls below paddle
